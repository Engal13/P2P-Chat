# Dear ImGui

Este directorio está destinado para alojar el código fuente de [Dear ImGui](https://github.com/ocornut/imgui).

Para compilar el proyecto correctamente, por favor copia aquí los siguientes archivos básicos del repositorio de Dear ImGui:

- `imgui.h`
- `imgui.cpp`
- `imgui_demo.cpp`
- `imgui_draw.cpp`
- `imgui_internal.h`
- `imgui_tables.cpp`
- `imgui_widgets.cpp`
- `imstb_rectpack.h`
- `imstb_textedit.h`
- `imstb_truetype.h`

Además de los archivos de backend (dependiendo de si vas a usar GLFW, SDL, OpenGL, DirectX, etc.), que comúnmente están en `backends/` del repo original (ej.: `imgui_impl_glfw.cpp`, `imgui_impl_opengl3.cpp`).
